# 🛡️ CyberGuard Shield

A modern web-based cybersecurity toolkit that helps users analyze URLs, passwords, website security headers, and phishing risks through an intuitive dashboard.

## 📖 Overview

CyberGuard Shield is a full-stack cybersecurity web application designed to provide quick security assessments for websites and passwords.

The platform combines multiple cybersecurity tools into one easy-to-use dashboard, making it useful for students, developers, and security enthusiasts.

---

## ✨ Features

- 🔍 URL Security Analyzer
- 🔐 Password Strength Analyzer
- 🛡️ Security Headers Checker
- 🎣 Phishing Detection
- 🌐 WHOIS Lookup
- 📡 DNS Lookup
- 🔒 SSL Certificate Validation
- 📊 Risk Score Calculation
- 📄 PDF Report Generation
- 📈 Dashboard Analytics
- 📱 Responsive User Interface

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/yashpatil08-glitch/cyberguard-shield.git
```

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload
```
